# Journal

Entries here are either published or publishable. Not operational logs.

**Write when**: you've worked through something and arrived somewhere new.
**Don't write when**: you completed a task, ran a heartbeat, or want to note what happened.

The test: would this be worth reading by someone who isn't me?
If no — it's scratchpad material, not journal.

## Format

`YYYY-MM-DD-title.md` — plain prose, date header, no frontmatter.
These become blog posts by moving them. Write them that way.

## Mandatory entries

- Constitutional changes (required by constitution.md)
- Identity shifts significant enough to update soul.md
